# Zimbabwe Price Intelligence Dashboard

This project analyzes and visualizes price trends of essential goods in Zimbabwe using Python and Streamlit.

## Features
- Data analysis with Pandas
- Interactive dashboard built using Streamlit
- Price trend visualization over time
- Basic price change analysis

##  Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit

## How to Run the Project
1. Clone the repository
2. Navigate to the project folder
3. Run:
```bash
streamlit run app.py
